
# 구글 앱스 스크립트(GAS) 설정 가이드

본 앱을 구글 스프레드시트와 완전히 연동하려면 다음 단계를 따르세요.

## 1. 스프레드시트 준비
1. 새로운 구글 스프레드시트를 만듭니다.
2. 첫 번째 행(Header)에 다음 항목을 입력합니다:
   `Date`, `URL`, `Thumbnail`, `Memo`, `Category`

## 2. 앱스 스크립트 작성
1. 스프레드시트 상단 메뉴: **확장 프로그램** > **Apps Script**를 클릭합니다.
2. `Code.gs` 파일에 제공된 `Code.gs` 내용을 복사하여 붙여넣습니다.
3. (선택) `DRIVE_FOLDER_ID`를 지정하면 이미지를 특정 폴더에 저장할 수 있습니다.

## 3. 배포 (Deployment)
1. 오른쪽 상단의 **배포** 버튼 > **새 배포**를 누릅니다.
2. 유형 선택: **웹 앱**
3. 설명: `Short-Form Bookmark App`
4. 웹 앱을 실행할 사용자: **나 (본인 계정)**
5. 액세스 권한이 있는 사용자: **모든 사용자 (Anyone)**
6. **배포** 버튼을 누르고 승인 절차를 거칩니다.

## 4. 프론트엔드 연결
1. 배포 후 제공되는 **웹 앱 URL**을 확인합니다.
2. 이 React 앱 코드에서 `google.script.run`을 통해 서버 함수(`getEntries`, `addEntry` 등)를 호출하게 됩니다.
   *주의: 본 샌드박스 환경에서는 Mock 데이터를 사용하지만, 실제 GAS `index.html`로 배포 시 `google.script.run`이 활성화됩니다.*

## 5. 자동 썸네일 기능
- 인스타그램/틱톡은 보안상 클라이언트(브라우저)에서 직접 이미지를 긁어오는 것이 차단(CORS)될 수 있습니다. 
- 이 앱은 **Gemini AI**를 활용하여 URL 분석을 보조하며, 실제 GAS 환경에서는 `UrlFetchApp`을 통해 서버사이드에서 OG 태그를 파싱하도록 구성되어 있습니다.
